Java Script Client Side X Server Side

O Java Script inicialmete era usado apenas no lado do cliente, porem agora possui frameworks que rodam no server side também.

Client-Server

Cliente - Servidor: O cliente (usurario) faz uma requisição ao servidor atravez do browser no qual o mesmo responde, seja com uma página HTML, um vídeo ou até mesmo um arquivo para download.

Um pouco de história

  • 1970: Arpanet
  • 1993: Tim Beners-Lee responsável pela criação da World Wide Web, linguagem HTML e o protocolo HTTP.
  • 1993: Surge o primeiro navegador: Mosaic
    • Google entra em cena
  • 1994: Surge o Netscape.
  • 1995: Brendan Eich: Estaria criando a linguagem livescrip, mas devido a popularidade do Java o nome foi alterado para JavaScript por puro marketing.
  • 1997: Netscape entrega o código do JS para a ECMA para a padronização da linguagem surgindo então o ECMAScript.
  • 2002: Após varias brigas entre Microsoft e Netscape, devido a popularização do internet explorer (não por escolha dos usurarios mas sim devido ao navegador vir por padrão no sistema operacional da Microsoft) a Netscape acaba falindo, mas não impede a criação da Mozila Foundation.
  • 2008: É nesse ano que a então famigerada Google entra em cena e ganha uma grande popularidade em apenas 2 anos.
  • 2009: Surge o motor V8 (JIT)
  • 2010: Surge o NodeJS possibilitando o uso do JavaScript do lado do server.
